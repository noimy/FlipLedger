⟳ Import from eBay
How to get your eBay CSV:
1. Go to eBay Seller Hub → Orders
2. Click Download report (top right) → select "Orders"
3. Download the file and select it below
📂 Click to select your eBay orders CSV
⚠ COGS will be blank — fill those in after import. All other fields auto-filled from eBay.
One-time setup (free):
1. Go to developer.ebay.com → Sign in with your eBay account
2. Create an app → get your User Token (OAuth)
3. Paste it below — it's saved locally, never sent anywhere
Pulls: order title, sale price, eBay fees, shipping, date. COGS still needs to be filled in manually.
